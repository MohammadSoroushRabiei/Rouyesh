# 💧 Ghadir_Watermark

یک اسکریپت ساده پایتون برای افزودن واترمارک (لوگو/نشان) به تصاویر به صورت دسته‌ای (Batch) با استفاده از کتابخانه OpenCV

> ⚠️ **توجه:** این پروژه صرفاً جهت نمایش و استفاده شخصی توسعه یافته و برای محیط Production بهینه‌سازی نشده است.

---

## ✨ ویژگی‌ها

- ✅ پردازش دسته‌ای تمام تصاویر موجود در یک پوشه
- ✅ تغییر سایز خودکار تصاویر ورودی به ابعاد استاندارد (۱۵۰۰×۱۰۰۰ پیکسل)
- ✅ قرار دادن واترمارک با کانال آلفا (شفافیت) در گوشه پایین-راست تصویر
- ✅ پشتیبانی از فرمت‌های رایج: `JPG`, `PNG`, `BMP`, `TIFF`
- ✅ ترکیب هوشمند تصاویر با حفظ کیفیت رنگ و جزئیات
- ✅ گزارش خطا برای تصاویر نامعتبر بدون توقف پردازش بقیه فایل‌ها

### 🔹 اجرای اسکریپت

از داخل پوشه `Ghadir_Watermark` دستور زیر را اجرا کنید:

```bash

```

✅ پس از اجرا، تصاویر پردازش‌شده در پوشه `Ghadir_WM/` ذخیره خواهند شد.

## ⚙️ تنظیمات سفارشی

پارامترهای قابل تغییر در فایل `code.py`:

| پارامتر | توضیح | مقدار پیش‌فرض |
|---------|--------|--------------|
| `input_folder` | پوشه تصاویر ورودی | `'Pictures/'` |
| `output_folder` | پوشه ذخیره خروجی | `'Ghadir_WM/'` |
| `watermark_path` | مسیر فایل واترمارک | `'Watermark_Small.png'` |
| `width` | عرض تصاویر خروجی | `1500` |
| `height` | ارتفاع تصاویر خروجی | `1000` |
| موقعیت واترمارک | آفست نسبت به گوشه پایین-راست | `20-` پیکسل عمودی، `50-` پیکسل افقی |

> 💡 **تغییر موقعیت واترمارک:**  
> برای جابجایی واترمارک، خط زیر را در کد ویرایش کنید:
> ```python
> watermark[-wm_h-20:-20, -wm_w-50:-50, :] = wm
> ```
> اعداد `-20` و `-50` را مطابق نیاز تغییر دهید.
